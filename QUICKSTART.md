# LMS Demo App - Quick Start

## What This Is

A complete Learning Management System (LMS) demo app with NO real database. Perfect for showcasing, presentations, or project demonstrations!

## Features

- ✅ Teacher Portal with dashboard, classes, assignments, submissions
- ✅ Student Portal with courses, assignments, attendance, grades
- ✅ Mock authentication (login with ANY email/password)
- ✅ Sample data pre-loaded (no database setup needed)
- ✅ Fully functional UI
- ✅ Ready to deploy to Vercel

## Quick Start (Local)

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Login

**Teacher Login:**
- Click "Enter as Teacher"
- Email: anything@example.com (or any email)
- Password: anything (or any password)

**Student Login:**
- Click "Enter as Student"  
- Email: student@example.com (or any email)
- Password: password (or any password)

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Deploy (NO environment variables needed!)
4. Done!
