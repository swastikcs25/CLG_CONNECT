## 🎯 Common Workflows

### As a Teacher (Full Flow):

1. **Login** → Dashboard
2. **My Courses** → Click "Create Course"
   - Name: "Computer Science 101"
   - Code: "CS101"
   - Credits: 3
   - Save
3. **Assignments** → Click "Create Assignment"
   - Title: "Homework 1"
   - Description: "Complete exercises"
   - Due date: Select date
   - Points: 100
   - Save
4. Wait for students to enroll and submit
5. **Mark Attendance** → Select "CS101"
   - Mark students present/absent/late
   - Save
6. **Submissions** → View student work
   - Enter grade (e.g., 85/100)
   - Add feedback: "Good work!"
   - Save

### As a Student (Full Flow):

1. **Login** → Dashboard
2. **My Courses** → See enrolled courses
   - View attendance percentage
3. **Attendance** → Check overall attendance
   - See present/absent/late counts
   - View history table
4. **Assignments** → See "Homework 1"
   - Click assignment
   - Upload file (PDF/DOC/image)
   - Submit
5. Wait for teacher to grade
6. **Assignments** → View grade
   - See score: 85/100
   - Read feedback: "Good work!"

## ✅ Quick Tips

1. **First time?** Create a teacher account first, then student account
2. **Teacher:** Create courses before marking attendance
3. **Student:** Enroll in courses to see them
4. **Files:** Max 25MB, PDF/DOC/Images supported
5. **Attendance:** Green ≥75%, Yellow ≥50%, Red <50%
6. **Refresh:** Changes appear in real-time

**Navigate confidently!** 🎯
